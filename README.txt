** TP_Git **
Projet de site web spécialisé sur les arbres

Description du projet :
La volonté de ce site est de présenter l'essentiel des variétés d'arbres en Europe.
Le site se veut interactif, laissant la possibilité aux visiteurs de laisser leurs propres photos, afin de constituer une banque d'images.

Liste des essences à ajouter :
 - chêne
 - hêtre
 - peuplier
 - érable
 - ...
